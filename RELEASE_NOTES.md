# mama-ai v1.1.0 Release Notes

This release upgrades the prototype into a hackathon-ready package with:

- CLI demo (`python -m mama_ai.src.main` or similar)
- FastAPI microservice (`uvicorn api:app --reload`)
- Streamlit UI demo (`streamlit run src/ui_demo.py`)
- Hackathon submission skeleton (`hackathon/` folder)
